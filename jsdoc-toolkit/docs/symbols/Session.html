<private> {Object} _provisionWithSession(request)
Provisiones the request with a session object. If the request has already a sessionId than it gets the session from the session store, otherwise it creates a new SessionId for the request and creates a new entry in the session store for the generated unique id.

handle(request, response, next)
Handles the HTTP request and sets a session for that request. It sets a cookie on the request if none is set it gets or creates a session for an existing cookie. It saves the session on the end of every request.
